# Note Thumbnail Studio - 仕様書 (Draft)

## 1. 概要
ブログサービス「note」の投稿用サムネイル（特に映画や小説の感想文用）を簡単に作成できるWebアプリケーション。
文字情報を中心としたデザインを、ブラウザ上で手軽に調整・画像化できることを目指す。

## 2. ターゲットユーザー
- noteでテキスト中心のコンテンツ（感想文、エッセイ等）を投稿するクリエイター
- 毎回PhotoshopやCanvaを開くのが手間に感じるユーザー

## 3. 基本機能要件

### 3.1 キャンバス設定
- **サイズ**: `1280 × 670 px` 固定（note推奨サイズ）。
- 将来的に他サイズ対応を考慮した設計とするが、初期リリースはこのサイズのみ。

### 3.2 テキスト編集機能 (Grid System)
- **要素制限**: 最大5つまで追加可能。
- **配置システム**:
    - 自由配置（Drag&Drop）ではなく、**定義されたグリッド/ゾーン**への配置を採用。
    - **レイアウト案**:
        - キャンバスを 3x3 または 垂直/水平分割 した領域（ゾーン）を定義。
        - 各テキスト要素に対し、「配置ゾーン（例：中央、左下、右上）」と「優先順位（1, 2, 3...）」を指定。
        - **スタッキング仕様**:
            - 同一ゾーン内の要素は **縦方向（Vertical）** に積み重なる。
            - 優先順位（リスト順）に従って上から下に配置される。
            - 要素間の間隔（Gap）は、ゾーンごとの設定または全体設定で調整可能にする（例: 狭い、普通、広い）。
    - これにより、誰が作っても整列された美しいレイアウトを保証する。
- **フォント設定**:
    - **日本語フォント対応**: Google Fonts等から日本語対応フォントを選定。
    - **拡張性**: フォントリストを定数/Config化し、容易に追加・変更できる設計にする。
    - 設定項目: サイズ、色、透明度、行間、文字間隔、太さ。

### 3.3 背景設定
- **初期**: 単色塗りつぶし（プリセットカラー + カラーピッカー）。
- **将来**: グラデーション、画像アップロード機能。

### 3.4 プレビュー & エクスポート
- テキスト編集と同時にプレビュー反映。
- PNG書き出し機能。

## 4. 技術スタック (確定)
- React (Vite)
- TypeScript
- Tailwind CSS (スタイリング)
- html2canvas (画像生成) - *要検証: 文字のレンダリング精度*
- GitHub Pages (デプロイ)

## 5. UI/UX コンセプト
- **Left Panel**: プレビュー（キャンバス）
- **Right Panel**: コントロール（テキスト編集、レイアウト選択、色設定）
- **グリッドガイド**: 編集時にグリッド線をオーバーレイ表示し、配置を視覚的に分かりやすくする。
